# Telegram-tool

--a tool to automate the process of adding members to your telegram groups.---

```
# Installation termux

$ apt update && apt upgrade
$ pkg install git
$ git clone https://www.github.com/0xbr0/Telegram-tool.git/
$ cd Telegram-tool
$ chmod +x install.sh
$ ./install.sh
$ python3 scrapper.py
$ python3 adder.py
```
